Project Overview

Developed a comprehensive VR medical training simulation for Neonatal Resuscitation Program (NRP) equipment handling. This hands-on, interactive training enables healthcare professionals to build confidence and procedural accuracy with all essential tools used during neonatal resuscitation emergencies.

Key Features

  • Complete NRP equipment catalog with interactive demonstrations
  • Monitoring devices: cardiac/respiratory monitor, BP cuffs, pulse oximeter
  • Airway tools: laryngoscope, ETT, PPV devices, suction systems
  • Thermal support: baby warmer, thermal mattress, NeoGuard
  • Medication & access: epinephrine syringes, UVC tray, umbilical clamps
  • Step-by-step equipment inspection and utilization
  • Photorealistic VR environment

My Contributions

  • Developed core interaction logic for all equipment types
  • Integrated equipment functionality and realistic animations
  • Created proper handling and sequencing mechanics
  • Implemented procedural guidance systems
  • Optimized performance for real-time simulation
