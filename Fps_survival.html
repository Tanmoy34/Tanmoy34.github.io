Project Overview

FPS Survival is a first-person wave defense shooter built in Unreal Engine 5 using C++. Dropped into an arena alone, the player must survive endless, escalating waves of spiders using a combination of responsive gunplay, tactical special abilities, and a point-driven economy system.

What sets this project apart is its ability system — three distinct time-manipulation and portal powers that demand strategic thinking rather than button mashing. Developed entirely solo, this project showcases proficiency in Unreal Engine C++ gameplay programming, AI wave management, and UI systems design.

Key Features

  • Smooth first-person shooting with responsive combat mechanics
  • Wave-based survival with escalating spider enemy difficulty
  • Time Slow ability — slows all enemies for a tactical window
  • Time Stop ability — freezes time completely to reposition or unload
  • Portal ability — spawns a damage-dealing portal to devastate incoming waves
  • Shop system — spend earned points between waves to purchase ammo
  • Point economy — earn points per kill, spend wisely to survive longer

Core Systems

  • Ability System — C++ cooldown timers with visual feedback for all three abilities
  • Wave Manager — dynamically scales spider count and speed each wave
  • Point & Shop Economy — kill-reward loop with between-wave purchasing

My Contributions

  • Built the wave spawning and difficulty scaling system
  • Developed the full point economy and shop system
  • Created HUD, shop widgets, and all UI elements
  • Solo-developed the project end-to-end from concept to playable build
