Project Overview

Developed a comprehensive VR-based interactive medical simulation focused on routine mouth and throat examination performed by family medicine doctors. This multiplayer-enabled simulation recreates a realistic clinical consultation environment with complete patient assessment capabilities.

Key Features

  • Complete oral cavity, lymph node, and thyroid gland examination
  • Realistic clinical consultation environment
  • Virtual patient communication and interaction
  • Multiplayer mode for collaborative training
  • Real-time instructor-student interaction
  • Built-in assessment system for performance tracking
  • Guided feedback throughout procedures

My Contributions

  • Developed examination procedure mechanics using C++
  • Implemented multiplayer networking for collaborative sessions
  • Created patient interaction and communication systems
  • Built assessment tracking and evaluation framework
  • Optimized performance for smooth multiplayer experience
