How to Play

  1. Start with 1,000 coins — manage your balance wisely.
  2. Choose a bet type from the four tabs: Outside Bets, Dozens, Columns, or a Straight Up number.
  3. Select a chip value (5, 10, 25, 50, or 100 coins) to set your denomination.
  4. Fine-tune your exact wager using the − / + buttons.
  5. Press ✦ SPIN ✦ and watch the wheel slow to a stop.
  6. Win based on where the ball lands — collect your payout or try again!
  7. If your balance hits zero, it resets to 1,000 coins automatically.

Project Overview

Royal Roulette is a fully playable casino-style roulette game built entirely from scratch with vanilla HTML5 Canvas, CSS3, and JavaScript — no libraries, no frameworks. It faithfully recreates the European Single Zero experience: a 37-pocket wheel drawn on Canvas with the authentic real-world number sequence, smooth eased spin animation, a complete 4-category betting table, casino chip selection, and a full payout system — all in a single HTML file.

Key Features

  • Authentic 37-pocket European wheel drawn on HTML5 Canvas
  • Real-world wheel pocket sequence (0, 32, 15, 19, 4…)
  • 4 bet categories — Outside, Dozens, Columns, Straight Up
  • 35:1 payout on single number and Green (0) bets
  • Casino chip system — 5 denominations (5 to 100 coins)
  • Smooth eased spin animation with 5–8 full rotations
  • Live results history strip (last 12 spins as coloured balls)
  • Win / Loss result modal after every spin
  • ⓘ Info modal with full how-to-play and payout table
  • Space key shortcut + Esc to close modals
  • Zero dependencies — single self-contained HTML file
