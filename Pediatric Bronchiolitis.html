Project Overview

Developed a multiplayer, open-scenario VR medical simulation for training healthcare professionals in recognizing and managing respiratory distress in infants. Centered around a 7-month-old patient, this simulation places learners in a realistic emergency department environment requiring critical thinking and teamwork.

Key Features

  • Realistic emergency department environment
  • Infant respiratory distress symptom assessment
  • Critical clinical decision-making scenarios
  • Multiplayer real-time collaboration
  • Effective communication training
  • Post-session debriefing system
  • Performance review and analysis tools

My Contributions

  • Developed multiplayer functionality for team collaboration
  • Implemented procedural event handling for dynamic scenarios
  • Created interactive VR elements for patient assessment
  • Built debriefing system for action review and learning
  • Ensured seamless, immersive learning experience
